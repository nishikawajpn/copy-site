# Usage

Webpackによって、コンパイル・トランスパイル・ミニファイをします。

まず、必要なパッケージをインストールします。

```bash
npm install
```

ローカルサーバーを立ち上げる

```bash
npm start
```

開発用に、コンパイル・トランスパイルする。

```bash
npm run build:dev
```

プロダクト用に、コンパイル・トランスパイル・ミニファイする。

```bash
npm run build
```
